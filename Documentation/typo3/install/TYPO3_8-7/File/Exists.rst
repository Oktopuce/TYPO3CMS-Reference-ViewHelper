===========
file.exists
===========


Simple view helper to check if given file is a regular file

= Examples =

<code title="Default">
<f:file.exists file="Absolute-path" />
</code>

<output>
TRUE or FALSE
</output>

Arguments
=========


.. _file.exists_file:
file
----

:aspect:`DataType`
   string

:aspect:`Required`
   true
:aspect:`Description`
   Absolute path
